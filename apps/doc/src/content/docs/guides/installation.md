`@rtorcato/js-common` is published on npm and works with any modern Node.js runtime.

## Requirements

- **Node.js** ≥ 22
- **TypeScript** ≥ 5.0 (optional, for TypeScript projects)

## Install

```bash
npm install @rtorcato/js-common
# or
pnpm add @rtorcato/js-common
# or
yarn add @rtorcato/js-common
```

## Use globally (CLI)

```bash
npm install -g @rtorcato/js-common
# or run ad-hoc
npx @rtorcato/js-common@latest --help
```
