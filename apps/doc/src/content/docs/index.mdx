## Why js-common?

<CardGrid stagger>
	<Card title="Ultra-lightweight" icon="rocket">
		Main bundle is only ~277 bytes. Individual modules add ~50–200 bytes each.
	</Card>
	<Card title="Tree-shakeable" icon="seti:folder">
		Named subpath exports — import only what you need.
	</Card>
	<Card title="TypeScript-first" icon="seti:typescript">
		Strict types, generics preserved, JSDoc-rich in your IDE.
	</Card>
	<Card title="CLI included" icon="seti:powershell">
		Run utilities directly from your terminal via `npx @rtorcato/js-common`.
	</Card>
</CardGrid>
