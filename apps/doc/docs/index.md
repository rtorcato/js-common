# js-common

A comprehensive set of common JavaScript and TypeScript utilities for Node.js projects.

- **Ultra-lightweight** — main bundle ~277 bytes; individual modules ~50–200 bytes each.
- **Tree-shakeable** — named subpath exports; import only what you need.
- **TypeScript-first** — strict types, generics preserved, JSDoc-rich in your IDE.
- **CLI included** — run utilities from your terminal via `npx @rtorcato/js-common`.

Start with [Installation](./guides/installation.md) or jump straight to the [Quick start](./guides/quick-start.md).
