## Exports

| --- | --- |

| `matchAll` | Returns all matches of a regex pattern in a string. |
| `replaceAllRegex` | Replaces all matches of a regex pattern in a string with a replacement. |
| `splitByRegex` | Splits a string by a regex pattern. |
